# Проекты для мобильных устройств

Этот репозиторий содержит различные проекты мобильных приложений, разработанные с использованием React Native и Expo.

## Проекты

### 1. Maps - 1 задание

Простое приложение на React Native, которое позволяет пользователям добавлять на карту маркеры и прикреплять к ним изображения.

**Ключевые технологии:**
*   React Native
*   Expo
*   React Native Maps
*   Expo Router
*   React Context

### 2. StickerSmash

Приложение для редактирования фотографий, позволяющее пользователям выбирать изображения из своей галереи и добавлять на них стикеры (эмодзи).

**Ключевые технологии:**
*   React Native
*   Expo
*   Expo Image Picker
*   React Navigation

### 3. Maps2 - 2 задание

Простое приложение на React Native, которое позволяет пользователям добавлять на карту маркеры и прикреплять к ним изображения, сохраняя данные в локальной базе данных SQLite.

**Ключевые технологии:**
*   React Native
*   Expo
*   React Native Maps
*   Expo Router
*   React Context
*   SQLite

### 4. Maps3 - 3 задание

Мобильное приложение на React Native, которое позволяет пользователям добавлять на карту маркеры с изображениями, отслеживать текущее местоположение и получать уведомления при приближении к сохранённым меткам.

**Ключевые технологии:**
*   React Native
*   Expo
*   React Native Maps
*   Expo Router
*   React Context
*   SQLite
*   Expo Location
*   Expo Notifications

---

Каждый проект находится в своей собственной директории и содержит свой собственный файл `README.md` с более подробной информацией.
