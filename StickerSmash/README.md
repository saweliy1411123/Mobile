# StickerSmash

Приложение React Native (Expo) для добавления стикеров на фотографии.

## Возможности

*   Выбор изображения из галереи.
*   Добавление различных эмодзи/стикеров на выбранное изображение.
*   Сброс изменений.
*   Сохранение отредактированного изображения (функционал будет реализован).
*   Навигация по вкладкам (Главная и О приложении).

## Быстрый старт

1.  **Установите зависимости:**
    ```bash
    npm install
    ```

2.  **Запустите приложение:**
    ```bash
    npm start
    ```

    Вы можете запустить приложение на Android, iOS или в веб-браузере.

## Структура проекта

*   `app/`: Основные экраны и навигация (Expo Router).
*   `assets/`: Изображения и ресурсы.
*   `components/`: Переиспользуемые UI-компоненты (например, `ImageViewer`, `EmojiList`, `Button`).
